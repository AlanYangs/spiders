### 基于SpringBoot实现的Java轻量级的web多线程爬虫框架

#### 框架模块包含但不限于：
- 免费代理构建
- 基于HttpClient的HTTP请求封装
- 多线程并发爬取
- 任务调度
- 免费云数据库模块以及集成了Mybatis支持入库MySQL数据库
